O Jogo do Nim

Joga contra o computador!

O Nim é um jogo de estratégia em que dois jogadores alternadamente removem objetos de diferentes pilhas. A cada jogada, o jogador deve remover pelo menos um objeto de uma pilha e todos os objetos removidos devem pertencer à mesma pilha. Ganha quem tirar o último objeto.

Antes de começar um jogo, o jogador deve escolher o tamanho do tabuleiro e a dificuldade do computador. Para remover peças de uma pilha, basta cliquar na bola a partir da qual os objetos desejados são eliminados.

No início do jogo, o jogador realiza o primeiro movimento, a menos que deixe o computador jogar primeiro pressionando o botão "Jogada do Computador".
